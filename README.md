# Motakadem Coptic Scouts

Arabic-first Next.js portal for a generic Coptic Orthodox scout church team.

## Implemented

- Next.js App Router + TypeScript + Tailwind
- Egyptian Arabic and English dictionaries in `src/i18n.ts`
- RTL Arabic default layout
- Redux Toolkit slices for auth, UI, locale, drafts, and filters
- RTK Query endpoints for events, RSVPs, notebook, roster, documents, and dashboard
- Scout feed, notebook, calendar, and profile pages
- Captain dashboard, event studio, document review, and roster pages
- Local visual assets for church scout events and avatars

## Scripts

- `npm run dev`
- `npm run build`
- `npm run typecheck`
- `npm run test`
- `npm run test:e2e`
- `npm run audit:content`
