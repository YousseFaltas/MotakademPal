# Motakadem Coptic Digital Core

This design system supports an Arabic-first Coptic Orthodox scout portal. The interface keeps the energetic blue scout dashboard style, but all spiritual tracking and sample content are now Coptic church appropriate.

## Identity

- Product name: `كشافة متقدم القبطية`
- Default locale: `ar-EG`
- Direction: RTL for Arabic, LTR for English
- Church identity: generic placeholders for parish, diocese, and patron saint

## Visual Style

- Navy/slate dark surfaces for focus.
- Cyan and blue for digital actions.
- Gold, ivory, burgundy, and olive for church/service accents.
- Horizontal gradients for primary actions and progress.
- Local event visuals replace external mockup URLs.

## Core Screens

- Scout feed with church scout events.
- Coptic spiritual follow-up notebook with private reflections.
- Calendar and profile/document status.
- Leader dashboard, event studio, document review, and roster.

## Privacy

Spiritual reflections are private to the scout. Leaders only see aggregate consistency.
